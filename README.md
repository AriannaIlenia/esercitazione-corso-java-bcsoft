# 📁 Esercitazione Java – OOP, Eccezioni, Enum e Collections

> Esercizi accademici sviluppati durante il corso Java presso BcSoft, focalizzati su programmazione orientata agli oggetti, gestione delle eccezioni, enum e collections.

## 📋 Descrizione

Questo progetto raccoglie una serie di esercizi pratici che coprono i concetti fondamentali della programmazione Java. Gli esercizi sono stati sviluppati in autonomia partendo da specifiche fornite dal corso.

## 🚀 Tecnologie utilizzate

| Tecnologia | Versione | Ruolo |
|---|---|---|
| Java | 17 | Linguaggio principale |
| Maven | 3.x | Gestione del progetto |

## 📌 Esercizi inclusi

| Classi | Concetti |
|---|---|
| `Programma` | ArrayList, Collections, Scanner |
| `Squadra`, `Giocatore` | OOP, List, metodi di business |
| `Lampadina`, `StatoLampadina`, `Interruttore` | Enum, gestione stati, composizione tra oggetti |
| `Dispositivo`, `Computer`, `Smartphone`, `SistemaOperativo` | Ereditarietà, polimorfismo, Enum |
| `UndiscoveredDimensionException` | Eccezioni personalizzate |

## ⚙️ Come avviarlo in locale

### Prerequisiti
- Java 17+
- STS o qualsiasi IDE Java

### Passi

```bash
git clone https://github.com/AriannaIlenia/esercitazione-corso-java-bcsoft.git
```
Importa il progetto nel tuo IDE ed esegui `Programma.java`.

## 💡 Cosa ho imparato

- Strutturazione di classi con responsabilità ben definite
- Utilizzo di `ArrayList` e lambda expressions
- Eccezioni personalizzate con `UndiscoveredDimensionException`
- `enum` per stati e categorie (StatoLampadina, SistemaOperativo)
- Polimorfismo con `Computer` e `Smartphone` che estendono `Dispositivo`
- Composizione tra oggetti (`Interruttore` che gestisce una `Lampadina`)
